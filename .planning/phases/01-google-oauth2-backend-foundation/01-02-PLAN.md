Create `src/routes/authRoutes.js` following the exact structure of `src/routes/userRoutes.js`:

```javascript
const express = require('express')
const router = express.Router()
const googleOAuthService = require('../services/googleOAuthService')
const userService = require('../services/userService')
const logger = require('../utils/logger')
const config = require('../../config/config')
const redis = require('../models/redis')
```

Implement two routes:

**GET /google** (AUTH-01: Initiate OAuth2 login flow):
1. Check `config.googleOAuth?.enabled` — if false, return 503
2. Check `config.userManagement?.enabled` — if false, return 503
3. Call `googleOAuthService.generateAuthUrl()` to get `{ authUrl, state }`
4. Store state in Redis with key `oauth_state:${state}` and TTL 600 seconds (10 min) using `redis.setex()`. Value can be JSON: `{ createdAt: new Date().toISOString(), ip: req.ip }`
5. Log: `logger.info('Initiating Google OAuth2 flow', { ip: req.ip })`
6. Redirect: `res.redirect(authUrl)`

**GET /google/callback** (AUTH-02, SYS-05: Handle callback):
1. Extract `code` and `state` from `req.query`
2. If no `code`, return 400: `{ error: 'Missing authorization code', message: 'Authorization code is required' }`
3. If `state` is present, validate against Redis key `oauth_state:${state}`. If not found, return 403: `{ error: 'Invalid state parameter', message: 'State validation failed. Please try logging in again.' }`. Delete the Redis key after validation (one-time use).
4. Call `googleOAuthService.exchangeCodeForTokens(code)` — wrap in try/catch. On failure, return 401: `{ error: 'Authentication failed', message: 'Failed to exchange authorization code' }`
5. Call `googleOAuthService.getUserProfile(tokenData.accessToken)` — on failure, return 401: `{ error: 'Authentication failed', message: 'Failed to retrieve user profile' }`
6. Call `googleOAuthService.validateDomain(profile.email)` — if false, return 403: `{ error: 'Domain not allowed', message: 'Your email domain is not authorized' }`
7. Map Google profile to user data: `{ username: profile.googleId, email: profile.email, displayName: profile.name, firstName: profile.givenName, lastName: profile.familyName, role: config.userManagement.defaultUserRole, isActive: true }`
8. Call `userService.createOrUpdateUser(userData)` to create or update user
9. Check `user.isActive` — if false, return 403: `{ error: 'Account disabled', message: 'Your account has been disabled' }`
10. Call `userService.recordUserLogin(user.id)`
11. Call `userService.createUserSession(user.id, { authMethod: 'google_oauth2', googleId: profile.googleId })`
12. Log success: `logger.info('Google OAuth2 login successful', { username: profile.googleId, email: profile.email, userId: user.id })`
13. Return 200: `{ success: true, message: 'Login successful', user: { id: user.id, username: user.username, email: user.email, displayName: user.displayName, firstName: user.firstName, lastName: user.lastName, role: user.role, picture: profile.picture }, sessionToken }`
14. Wrap entire callback handler in outer try/catch for unexpected errors — return 500: `{ error: 'Authentication error', message: 'Internal server error during authentication' }`

Export: `module.exports = router`

IMPORTANT: Follow codebase conventions — no semicolons, single quotes. Run `npx prettier --write` after creating.
